# @datafire/amazonaws_inspector

Client library for Amazon Inspector

## Installation and Usage
```bash
npm install --save datafire @datafire/amazonaws_inspector
```

```js
let datafire = require('datafire');
let amazonaws_inspector = require('@datafire/amazonaws_inspector').actions;

let account = {
  hmac: "",
}
let context = new datafire.Context({
  accounts: {
    amazonaws_inspector: account,
  }
})


amazonaws_inspector.AddAttributesToFindings({}, context).then(data => {
  console.log(data);
})
```

## Description
<fullname>Amazon Inspector</fullname> <p>Amazon Inspector enables you to analyze the behavior of your AWS resources and to identify potential security issues. For more information, see <a href="http://docs.aws.amazon.com/inspector/latest/userguide/inspector_introduction.html"> Amazon Inspector User Guide</a>.</p>

## Actions
### AddAttributesToFindings
Assigns attributes (key and value pairs) to the findings that are specified by the ARNs of the findings.


```js
amazonaws_inspector.AddAttributesToFindings({
  "body": {
    "findingArns": [],
    "attributes": []
  },
  "Action": "",
  "Version": ""
}, context)
```

#### Parameters
* body (object) **required**
* Action (string) **required**
* Version (string) **required**
* X-Amz-Content-Sha256 (string)
* X-Amz-Date (string)
* X-Amz-Algorithm (string)
* X-Amz-Credential (string)
* X-Amz-Security-Token (string)
* X-Amz-Signature (string)
* X-Amz-SignedHeaders (string)

### CreateAssessmentTarget
Creates a new assessment target using the ARN of the resource group that is generated by <a>CreateResourceGroup</a>. You can create up to 50 assessment targets per AWS account. You can run up to 500 concurrent agents per AWS account. For more information, see <a href="http://docs.aws.amazon.com/inspector/latest/userguide/inspector_applications.html"> Amazon Inspector Assessment Targets</a>.


```js
amazonaws_inspector.CreateAssessmentTarget({
  "body": {
    "assessmentTargetName": "",
    "resourceGroupArn": ""
  },
  "Action": "",
  "Version": ""
}, context)
```
