# @datafire/npr_rss

Client library for www.npr.org

## Installation and Usage
```bash
npm install --save @datafire/npr_rss
```
```js
let npr_rss = require('@datafire/npr_rss').create();

npr_rss.news({}).then(data => {
  console.log(data);
})
```
